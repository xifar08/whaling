# Whaling

Ce projet se concentre sur l'exploration de plusieurs jeux de données disponibles sur le site : <https://whalinghistory.org/>

Les données traitent de la pèche à la baleine par des navires américains.

Plusieurs cartes peuvent être générées grâce au notebook :

- [map_alea](maps/map_alea.html)map_alea : la représentation d'un voyage au hasard
- map_choisi : la représentation d'un voyage en particulier
- map_sight : la représentation de tous les endroits où une baleine a été aperçue.
